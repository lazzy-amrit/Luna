# 📦 Installation

```bash
git clone https://github.com/lazzy-amrit/Luna.git

cd luna

pip install -r requirements.txt
```

Create a `.env` file:

```env
BOT_TOKEN=YOUR_DISCORD_BOT_TOKEN
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Run:

```bash
python bot.py
```
